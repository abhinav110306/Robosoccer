# IEEE Robosoccer Robot

This project involved the design and development of a robosoccer robot platform for competitive robotic soccer.

## Contributions
- Designed and fabricated the robot chassis using Autodesk Fusion 360
  
## Chassis Fabrication

The robot chassis was fabricated from HDPE (High-Density Polyethylene) due to its lightweight, durability, and impact resistance.

Manufacturing steps included:
- Cutting HDPE sheets to required dimensions using a hand saw
- Drilling mounting and assembly holes using a drill press / drilling machine
- Mechanical assembly of drivetrain components onto the chassis

This fabrication approach enabled rapid prototyping and easy modification of the robot platform during development.

## Tools Used
- Autodesk Fusion 360
- CAD modeling and assembly design
